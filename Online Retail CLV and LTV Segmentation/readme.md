# CLV
## 01 什么是客户终身价值？（下文将以简写CLV代替）
定义： CLV是客户现在以及未来所有产生的现金流的净现值 - 公司在该客户身上的所有支出（广告支出、retention支出等）

## 02 CLV模型能用来做什么？
对于公司来说，用于衡量：acquire or retain一位客户值得花费多少钱
